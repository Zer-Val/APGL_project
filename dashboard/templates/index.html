Final project in APGL - Dashboard of Tesla's stock price

This dashboard is the final project for the Advanced Python, Git, Linux for Bloomberg course of ESILV, Year 4. The goal was to do a dashboard that runs on a web hosted server. It runs on Linux, and scrapes data from the website Google Finance and get the stock value of Tesla. The scraper runs every 5 minutes and a graph is displayed with the values. Furthermore, every day at 8 PM (UTC-4), a report is produced to analyse the data of the day and displayed on a table beneath. The report is bash generated, and the update of the table is done automatically.
